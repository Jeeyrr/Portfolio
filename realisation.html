GestEPI

📆 Durée : 01/02/2025 au 15/04/2025

🎯 Objectif : Développer une application de gestion de stocks et EPI (Équipements de Protection Individuelle) pour une entreprise simulée.

📦 Contexte : Projet réalisé en formation dans le cadre du BTS SIO SLAM. Cette application permet de suivre les stocks, gérer les attributions d'équipements et générer des statistiques d'utilisation.

React TypeScript SQL Node.js WAMP MariaDB npm

⚙️ Fonctionnalités principales :

  • Gestion du stock d'équipements de protection
  • Attribution d'EPI aux employés
  • Suivi des dates de péremption et renouvellements
  • Statistiques d'utilisation et rapports
  • Interface utilisateur intuitive et responsive

Projet API : Ville Lumière (Ligue 1)

📆 Durée : Septembre 2024 – Décembre 2024

🎯 Objectif : Créer une plateforme d'information sur la Ligue 1 en exploitant les données de l'API-FOOTBALL. L'objectif était d'afficher des infos actualisées sur les équipes, classements et résultats de match via une API externe, dans un site construit avec Django.

📦 Contexte : Projet d'équipe réalisé en 2ème année de BTS SIO SLAM. L'idée est née d'une passion commune pour le football. Le site permet d'explorer dynamiquement les données sportives de la Ligue 1 à travers une API, tout en apprenant à structurer un projet Django complet.

HTML CSS JavaScript Python Django API-FOOTBALL

⚙️ Fonctions principales :

  • Affichage du classement en temps réel de la Ligue 1
  • Actualités et dernières infos football
  • Pages dédiées aux équipes (logo, statistiques, etc.)
  • Appels API pour récupérer les résultats de matchs
  • Rendu dynamique des données avec Django + templates
